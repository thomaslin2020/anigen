# Anigen

To try out our code, just type `python train.py`

Make sure you have CUDA enabled for your computer.

Libraries used:
- Gdown
- Matplotlib
- PyTorch
- Scipy
- Torchvision

Install Gdown: `pip install gdown`
Install Matplotlib: `pip install matplotlib`
Install PyTorch: Visit https://pytorch.org/get-started/locally/.
Install Scipy: `pip install scipy`
Install Torchvision: `pip install torchvision`

More updates, including comments and a more organized code structure, will be added later. So stay tuned!
